About

Mazaar Framework is free PHP 5.0 library of classes which provides wide range of tools for website building. Is amied on skilled PHP programmators who looking for some develoment tools which made their work easier.

In final state there will be tools for modeling site in WebML, installators for packages, unifyied content managing and others.

Development notes

Mazaar framework consists from core and extending packages. Core contains only set of classes on which you can build up you site.

Core sources

  • Debug.inc - provides simple debug servicies,
  • Exception.inc - handling scripts exceptions,
  • Preferences.inc - handling Mazaar preferences,
  • Mazaar.inc - Main file. Contains Mazaar, Package classes and their exeptions classes (MazaarException, PackageException),
  • Mazaar.xml - main configuration file.

Packages

Features of Mazaar framework could be extended by additional packages, which are built up on top of Package class.

There are the group of generic packages which are already done. It provides common servicies for some site types.
